# Accountability — beta releases

Personal focus tracker for macOS. Beta builds for friends.

## Install (or update)

Paste in Terminal:

```sh
curl -fsSL https://raw.githubusercontent.com/paulthewebdeveloper/accountability-releases/main/install.sh | sh
```

The app also updates itself when a new build is out.

Everything the app sees stays on your Mac — screen contents, window titles
and activity never leave it. Optional account sync covers settings, rules
and daily totals only.
